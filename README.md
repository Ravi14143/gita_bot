# Vite React TypeScript Starter

A fast frontend setup using **Vite**, **React**, **TypeScript**, and **Tailwind CSS**.

## 🚀 Setup

```bash
npm install
npm run dev
````

Then open [http://localhost:5173](http://localhost:5173)

## 🏗️ Build

```bash
npm run build
```

## 🔍 Preview

```bash
npm run preview
```

## 🧹 Lint

```bash
npm run lint
```

## 📁 Stack

* Vite + React + TypeScript
* Tailwind CSS
* ESLint + Lucide Icons
